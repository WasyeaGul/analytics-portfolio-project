Description:
This dataset contains information on credit card customers, including
demographics, credit limits, payment history, bill amounts, and whether
a customer defaulted on their payment.

Use Case:
The dataset is used to analyze credit risk and identify factors associated
with customer default behavior in a banking context.

Target Variable:
- default_payment_next_month (1 = default, 0 = non-default)
