# Analytics Portfolio Project

## Project Overview
End-to-end SQL + Python analytics project focused on data exploration, modeling, and business insights.

## Tools & Technologies
- Python (pandas, numpy, matplotlib, scikit-learn)
- SQL
- Jupyter Notebooks

## Project Structure
- `data/` – Dataset information and access instructions
- `sql/` – SQL queries for data extraction and transformation
- `notebooks/` – Python analysis and modeling notebooks
- `src/` – Reusable Python scripts

## Key Questions
- What patterns exist in the data?
- What factors influence outcomes?
- How can insights support better decision-making?

In progress
